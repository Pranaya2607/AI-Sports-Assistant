# Project Report Summary

## Title

AI-Based Sports Equipment Recognition and Recommendation System using CNN and Generative AI

## Abstract

The proposed system identifies sports equipment from images using a CNN and generates useful recommendations using Gemini AI and RAG. The system is built using React, FastAPI, PyTorch, FAISS, Sentence Transformers, and Google Gemini API.

## Key Contributions

- Sports equipment recognition using MobileNetV3 Large.
- RAG-based knowledge retrieval from PDFs and text files.
- Gemini-powered recommendation and assistant features.
- Full-stack implementation with modern UI.
- Multiple practical modules for size, maintenance, condition, and fake detection.
